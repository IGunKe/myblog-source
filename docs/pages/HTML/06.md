# 表格与多媒体

|属性	|说明			|
|caption|表格标题		|
|thead	|表头部分		|
|tbody	|表格主体部分	|
|tfoot	|表格尾部		|
```html
<table border="1">
        <caption>后盾人表格标题</caption>
        <thead>
            <tr>
                <th>标题</th>
                <th>时间</th>
            </tr>
        </thead>
        <tbody>
            <tr>
                <td>后盾人</td>
                <td>2020-2-22</td>
            </tr>
        </tbody>
</table>
```
