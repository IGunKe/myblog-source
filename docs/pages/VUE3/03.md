# 响应式ref全家桶

## ref
```html
<script setup>
import {ref} from 'vue';
//ref中放入一个基础类型数值，会返回一个响应式的refImpl对象
const message = ref('message');

console.log(message);
</script>
```
## isRef
```html
<script setup>
import {ref, isRef} from 'vue';
//ref中放入一个基础类型数值，会返回一个响应式的refImpl对象
const message = ref('message');
// isRef判断是不是一个ref对象

console.log(isRef(message));//true
</script>
```
## shallowRef
创建一个跟踪自身 .value 变化的 ref，但不会使其值也变成响应式的
```html
<script setup>
import {ref, isRef, shallowRef} from 'vue';
//ref中放入一个基础类型数值，会返回一个响应式的refImpl对象
const message = ref('message');
// isRef判断是不是一个ref对象
console.log(isRef(message));//true
// shallowRef
const myName = shallowRef({
    name: 'xdj'
})
const changeName = () => {
    myName.value = {name: 'hello'};
}
console.log(myName.value);
</script>

<template>
   <div>
    <button @click="changeName">change</button>
    <div>{{myName}}</div>
   </div>
</template>
```
## triggerRef
triggerRef强制更新页面DOM
```html
<script setup>
import {ref, isRef, shallowRef, triggerRef} from 'vue';
// shallowRef
const myName = shallowRef({
    name: 'xdj'
})
const changeName = () => {
    //不会更新dom
    myName.value.name = 'hello';
    // triggerRef强制更新页面DOM
    triggerRef(myName);
}
</script>

<template>
   <div>
    <button @click="changeName">change</button>
    <div>{{myName}}</div>
   </div>
</template>
```
## customRef
自定义ref ,customRef 是个工厂函数要求我们返回一个对象 并且实现 get 和 set  适合去做防抖之类的
```html
<script setup>
import {customRef} from 'vue';
function myRef(value) {
    let timer;
    return customRef((track, trigger) => {
        return {
            get() {
                track();
                return value;
            },
            set(newVal) {
                clearTimeout(timer);
                timer = setTimeout(() => {
                    console.log('触发了set');
                    value = newVal;
                    trigger();
                }, 500);
            }
        }
    })
}
const name = myRef('xdj');
const changeName = () => {
    name.value = 'hello';
}
</script>

<template>
   <div>
    <button @click="changeName">change</button>
    <div>{{name}}</div>
   </div>
</template>
```
