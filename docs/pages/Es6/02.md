# 02 变量的解构赋值
## 1.数组的解构赋值
从数组和对象中提取值，对变量进行赋值，这被称为解构
```js
let [a, b, c] = [1, 2, 3];
console.log(a);// 1
```
本质上，属于“模式匹配”，只要等号两边的模式相同，左边的变量就会被赋予对应的值
```js
let [foo, [[bar], baz]] = [1, [[2], 3]];
foo // 1
bar // 2
baz // 3

let [ , , third] = ["foo", "bar", "baz"];
third // "baz"

let [x, , y] = [1, 2, 3];
x // 1
y // 3

let [head, ...tail] = [1, 2, 3, 4];
head // 1
tail // [2, 3, 4]

let [x, y, ...z] = ['a'];
x // "a"
y // undefined
z // []
```
另一种情况是不完全解构，即等号左边的模式，只匹配一部分的等号右边的数组。这种情况下，解构依然可以成功
```js
let [x, y] = [1, 2, 3];
x // 1
y // 2

let [a, [b], d] = [1, [2, 3], 4];
a // 1
b // 2
d // 4
```
如果等号的右边不是数组（或者严格地说，不是可遍历的结构，参见《Iterator》），那么将会报错
```js
// 报错
let [foo] = 1;
let [foo] = false;
let [foo] = NaN;
let [foo] = undefined;
let [foo] = null;
let [foo] = {};
```
 Set 结构，也可以使用数组的解构赋值
 ```js
let [x, y, z] = new Set(['a', 'b', 'c']);
x // "a"
```
### 默认值
解构赋值允许指定默认值。
```js
let [foo = true] = [];
foo // true

let [x, y = 'b'] = ['a']; // x='a', y='b'
let [x, y = 'b'] = ['a', undefined]; // x='a', y='b'
```
## 2.对象的解构赋值

```js
let {foo, bar} = {foo: 'aaa', bar: 'bbb'};
foo //aaa
bar //bbb
```
对象的解构也可以指定默认值

```js
var {x = 3} = {};
x // 3
var {x, y = 5} = {x: 1};
x // 1
y // 5

var {x: y = 3} = {};
y // 3

var {x: y = 3} = {x: 5};
y // 5
```
## 3.字符串的解构赋值
```js
const [a, b, c, d, e] = 'hello';
a // "h"
b // "e"
c // "l"
d // "l"
e // "o"
```
